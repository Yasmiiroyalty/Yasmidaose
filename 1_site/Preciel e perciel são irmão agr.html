Elementos HTML

    Main root

    <html>- O elemento HTML representa a raiz (elemento de nível superior) de um documento HTML, portanto também é referido como o elemento raiz. Todos os outros elementos devem ser descendentes desse elemento.

    Document metadata

    <link>- O elemento HTML externo recurso Link (ligação) especifica as relações entre o documento atual e um recurso externo. Este elemento é mais comumente usado para vincular a folhas de estilo, mas também é usado para estabelecer ícones local ("favicon" ícones de estilo e ícones de tela/app mobile home), entre outras coisas.

    <meta>- O elemento HTML meta representa metadados que não podem ser representados por outros elementos HTML meta-relacionados, como base, link, script, style ou title.

    <style>- O elemento HTML style contém informações de estilo para um documento ou parte de um documento.

    <title>- O elemento HTML Title (title) define o título do documento que é mostrado na barra de título do navegador ou guia de uma página.

    Sectioning root

    <body>- O HTML corpo elemento representa o conteúdo de um documento HTML. Pode haver apenas um elemento body em um documento.

    content sociating

    <adress>- O elemento Html address indica a inclusão do HTML, provencia os contacta as informações para pessoas ou organizações.

    <article> - O elemento Html article respresenta contaminação própria de um documento, página,aplicação ou no site, no qual é desejado para ser independente distribuível ou reatulizável. Exemplos incluidos: Postagen fórum, magazine ou jornais ou blog de entrada.

    <aside>- O elemento Html aside representa uma parte de um documento cujo conteúdo só indiretamente está relacionado com o conteúdo do documento principal.

    <footer>- O elemento Html footer representa um rodapé para o seu conteúdo de corte mais próximo ou a secionar o elemento raiz. Normalmente, um rodapé contém informações sobre o autor da secção, dados de direitos autorais ou links para documentos relacionados.

    <header>- O elemento Html header representa o conteúdo introdutório, normalmente um grupo de "aids" introdutório ou navegação. Pode conter alguns elementos de título, mas também outros elementos como um logotipo, um formulário de pesquisa, um nome de autor e assim por diante.

    <h1,h2,h3,h4,h5,h6>- O HTML h1 – h6 elementos representam seis níveis de títulos de seção. h1 é o mais alto nível de seção e h6 é o mais baixo.

    <hgroup>- O elemento HTML hgroup representa uma rubrica multi-nível para uma seção de um documento. Agrupa um conjunto de h1– h6 elementos.

    <p>- O elemento HTML principal representa o conteúdo dominante do corpo de um documento, parte de um documento ou aplicativo. Área de conteúdo principal consiste de conteúdo que está diretamente relacionado à ou expande-se sobre o tema central de um documento, ou a funcionalidade central de um aplicativo.

    <nav>- O elemento nav HTML representa uma seção de uma página cujo objetivo é fornecer links de navegação, ou dentro do documento atual ou a outros documentos. Exemplos comuns de seções de navegação são menus, tabelas de conteúdo e índices.

    <section>- O elemento HTML section representa uma seção autônoma — que não tem um elemento semântico mais específico para representá-lo contido em um documento HTML.

    Inline text semantics

    <a>- Ele HTML elemento (ou elemento âncora) cria um hiperlink para outras páginas da web, arquivos, locais dentro da mesma página, endereços de e-mail ou qualquer outro URL.

    <abbr>- O elemento de abreviação HTML (abbr) representa uma abreviação ou acrônimo; o atributo de título opcional pode fornecer uma expansão ou descrição para a abreviação.

    <b>- O elemento HTML Bring Attention To (b) é usado para chamar a atenção do leitor para o conteúdo do elemento, que de outra forma não recebe importância especial.

    <bdi>- O elemento HTML Bidirectional Isolate () informa ao algoritmo bidirecional do navegador para tratar o texto que ele contém isoladamente de seu texto ao redor.

    <bdo>- O elemento Substituir texto bidirecional de HTML () substitui a direcionalidade atual do texto, para que o texto dentro seja renderizado em uma direção diferente.

    <br>- O elemento HTML
    produz uma quebra de linha no texto (retorno de carro). É útil para escrever um poema ou um endereço, onde a divisão de linhas é significativa.

    <cite>- The HTML Citation element () is used to describe a reference to a cited creative work, and must include either the title or author or the URL of that work.

    <cote>- O elemento HTML exibe seu conteúdo estilizado de forma a indicar que o texto é um pequeno fragmento de código de computador.

    <data>- O elemento HTML vincula um determinado conteúdo a uma tradução legível por máquina. Se o conteúdo estiver relacionado a hora ou data, o elemento deve ser usado.

    <dfn>- O elemento HTML Definition () é usado para indicar o termo que está sendo definido no contexto de uma frase ou frase de definição.

    <em>- O elemento HTML marca o texto que tem ênfase de ênfase. O elemento pode ser aninhado, com cada nível de aninhamento indicando um maior grau de ênfase.

    <i>- O elemento HTML representa um intervalo de texto que é separado do texto normal por algum motivo. Alguns exemplos incluem termos técnicos, frases de idiomas estrangeiros ou pensamentos de personagens fictícios. Normalmente é exibido em tipo itálico.

    <Kbd>- O elemento HTML Keyboard Input () representa um intervalo de texto embutido que indica a entrada textual do usuário a partir de um teclado, entrada de voz ou qualquer outro dispositivo de entrada de texto.

    <mark>- O elemento HTML Mark Text () representa o texto marcado ou destacado para fins de referência ou notação, devido à relevância ou importância da passagem marcada no contexto de inclusão.

    <q>- O elemento HTML indica que o texto em anexo é uma cotação curta em linha. A maioria dos navegadores modernos implementam isso cercando o texto entre aspas.

    <rb>- O elemento HTML Ruby Base () é usado para delimitar o componente de texto base de uma anotação , ou seja, o texto que está sendo anotado.

    <rp>- O elemento Parêntese de Fallback de Ruby HTML () é usado para fornecer parênteses de retorno para navegadores que não suportam a exibição de anotações de ruby usando o elemento anotado.

    <rt>- O elemento HTML Ruby Text () especifica o componente de texto rubi de uma anotação ruby, que é usada para fornecer informações de pronúncia, tradução ou transliteração para tipografia do Leste Asiático. O elemento deve estar sempre contido em um elemento .

    <rtc>- O elemento HTML Ruby Text Container () abrange anotações semânticas de caracteres apresentados em um rubi de elementos usados dentro do elemento . Os elementos podem ter as duas anotações de pronúncia () e semântica ().

    <ruby>- O elemento HTML representa uma anotação ruby. Anotações Ruby são para mostrar pronúncia de caracteres do leste asiático.

    <s>- O elemento HTML renderiza texto com uma riscagem ou uma linha através dele. Use o elemento para representar coisas que não são mais relevantes ou que não são mais precisas. No entanto, não é apropriado ao indicar edições de documentos; para isso, use os elementos e , conforme apropriado.

    <samp>- O Elemento de Amostra HTML () é usado para incluir texto embutido que representa a saída de amostra (ou cotada) de um programa de computador.

    <small>- O elemento HTML torna o tamanho da fonte do texto um tamanho menor (por exemplo, de grande para médio, ou de pequeno para x-pequeno) até o tamanho de fonte mínimo do navegador. Em HTML5, esse elemento é redefinido para representar comentários secundários e letras pequenas, incluindo direitos autorais e texto jurídico, independentemente de sua apresentação estilizada.

    <span>- O elemento HTML é um contêiner in-line genérico para conteúdo de frases, que não representa nada inerentemente. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id) ou porque eles compartilham valores de atributos, como lang.

    <strong>- O Elemento de Importância Forte de HTML () indica que seu conteúdo tem forte importância, seriedade ou urgência. Os navegadores normalmente renderizam o conteúdo em negrito.

    <sub>- O elemento Subscrito HTML () especifica o texto embutido que deve ser exibido como subscrito apenas por motivos tipográficos.

    <sup>- O elemento HTML Superscript () especifica o texto embutido que deve ser exibido como sobrescrito apenas por motivos tipográficos.

    <time>- The HTML element represents a specific period in time. It may include the datetime attribute to translate dates into machine-readable format, allowing for better search engine results or custom features such as reminders.

    <tt>- O elemento de texto de teletipo HTML obsoleto () cria texto embutido que é apresentado usando a face de fonte monoespaçada padrão do agente do usuário.

    <u>- O elemento HTML Unarticulated Annotation () representa um intervalo de texto embutido que deve ser renderizado de forma a indicar que ele possui uma anotação não textual.

    <var>- O elemento HTML Variable () representa o nome de uma variável em uma expressão matemática ou em um contexto de programação.

    <wbr>- O elemento HTML representa uma oportunidade de quebra de palavra - uma posição no texto em que o navegador pode, opcionalmente, dividir uma linha, embora suas regras de quebra de linha não criassem uma quebra nesse local.
